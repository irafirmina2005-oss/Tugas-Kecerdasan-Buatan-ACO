# Tugas-Kecerdasan-Buatan-ACO
Program untuk menyelesaikan traveling salesman problem mengenai Ant Colony Optimization
Program menggunakan Ant Colony Optimization dengan pembatasan bahwa node yang telah dikunjungi tidak boleh dikunjungi kembali.
- Referensi: Raghavendra BV (2015). Solving Traveling Salesmen Problem using Ant Colony Optimization Algorithm. J Appl Computat Math 4: 260.

# Parameter ACO yang digunakan:
- Jumlah Semut : 20
- Iterasi : 100
- Alpha : 1
- Beta : 3
- Evaporation Rate : 0.5
- Q : 100

- Titik Awal: H
- Titik Akhir: D
- Titik Wajib Dilewati: #
